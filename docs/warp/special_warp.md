# Special Warp

warpの中でも特別な扱いを受けているwarp

fly warp, dungeon warp, その他 の3種類に大別できる

special warpのwarp情報は `data/special_warp.asm` で定義されている

## Fly warp

ワープパネル、『あなぬけのひも』、『あなをほる』、『テレポート』、『そらをとぶ』によるマップ移動
