# カートリッジ情報

英語版

 - カートリッジタイプ: ROM + MBC3 + RAM + Battery
 - ROM容量: 1MB (16KB * 64バンク)
 - RAM容量: 32KB (8KB * 4バンク)

日本語版

 - カートリッジタイプ: ROM + MBC1 + RAM + Battery
 - ROM容量: 512KB (16KB * 32バンク)
 - RAM容量: 32KB (8KB * 4バンク)

ちなみにピカチュウ版は日本語ROMがMBC3、英語ROMがMBC5とポケモン赤とは異なる。
